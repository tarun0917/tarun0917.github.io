Autonomous Mobile Robot

The objective of this project is to create an autonomous mobile robot that follows the track. An Android smartphone is used as a camera and the interactive user interface. The phone also sends information to a PIC32 microchip to control the motors' steering wheels. A user can adjust the tracked color threshold through the Android application on the phone.



The chassis was designed using Onshape and were constructed using acrylic sheets. The acrylic sheets were cut to shape using a laser cutter and a few parts such as the motor mounts were 3d printed since they are more durable.

The arduino was connected to the PIC32 via USB CDC and the an application was designed were the camera reads the center of the line and sends back the co-ordinates to the PIC32.

The PIC32 reads the co-ordinates and then sends a pwm signal to the motor which in turn centers the robot along the center of the line. The robot was battery operated with a Lipo battery.
