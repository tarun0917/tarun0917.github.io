Motor Control Project

The main goal of this project is to create a simple user interface i.e a list of commands that gives us complete control of the motor. This project consists of two parts, the hardware and the software part. The major hardware components that was used for this project are a brushed DC motor, microcontroller, MAX9918 current-sense amplifier, DRV8835 H-bridge and a battery pack. The basic outline of the hardware connections are shown as below :


Once the harware was developed, it was time to develop the software. The motivation for developing this project is to build an intelligent motor driver where the user has the ability to control it by simply clicking a button. This algorithm provides the user with a simple user interface: a list of commands one can chose by pressing a key. Following are the list of commands available for the user :


The overall flow of the algorithm is shown as below :


There are two control loops for this algorithm. The inner current control loop and the outer position control loop. First the gains was set for the inner current control loop using a preset wave (square waveform) and then the gains were set for the outer position control loop.

The type of trajectories that were developed for this project are a step trajectory and a cubic trajectory. The user first sets the control gains by using option 'i' and then loads the step or cubic trajectory using option 'm' or 'n'. Finally option 'o' is selected to execute the trajectory and a plot is obtained which shows the actual path and the path taken by the motor, the error can be calculated from the this plot.
